By setting the closeOthers input on NgkAccordionComponent to true, when an accordion panel is selected the others will automatically be collapsed.

By setting the isOpenOnInit input on an NgkAccordionPanelComponent to true, that panel will start as expanded. Note that panel #1 is expanded in the exmaple.
