By setting the closeOthers input on XmAccordionComponent to true, when an accordion panel is selected the others will automatically be collapsed.

By setting the isOpenOnInit input on an XmAccordionPanelComponent to true, that panel will start as expanded. Note that panel #1 is expanded in the exmaple.
