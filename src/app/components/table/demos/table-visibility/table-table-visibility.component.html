The table column visibility can be set using the colVisibility input, which accepts an array of boolean values (true for visible, false for hidden). The colVisibility input does not remove the column from the dom like *ngIf does (for *ngIf dom control of columns, see the colWidth input on the <th xm> below).

WARNING: If using colVisibility on the <table xm> tag, table columns cannot be hidden/removed using *ngIf on the <th> and <td> tags. For *ngIf dom control of columns, set widths using the colWidth input on the <th xm> tags.

The colWidths values are the same as the previous example. Some additional styling has been applied to affect the background and padding.

*NOTE: If the 'Username' column is removed, as none of the other columns have a flex grow greater than 0, the table does not fill the whole area.
