The table column visibility can be controlled using *ngIfs placed on the <th> and <td> tags if the column widths are set using colWidth inputs on the <th xm> tags.

If setting widths using the colWidth inputs on the <th xm> tags, column visibility can still be controlled using the colVisibility input on the <table xm> tag. Best practice is to use either the colVisibility or *ngIfs, but not both.

WARNING:Using both the colWidths on the <table xm> tag and colWidth inputs on the <th xm> tags is redundant, unnecessary, and will likely result in unintended behavior. Choose one or the other.
