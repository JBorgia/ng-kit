The table column widths can be set using the colWidths input on the <table ngk> tag, which accepts an array of type NgkColWidth objects. Type NgkColWidth objects are flex-based and have 'grow', 'shrink', and 'basis' fields.

If setting widths using the colWidths input, column visibility should be controlled using the colVisibility input. See the next demo for details.

WARNING: If using colWidths on the <table ngk> tag, table columns cannot be hidden/removed using *ngIf on the <th> and <td> tags. For *ngIf dom control of columns, set widths using the colWidth input on the <th ngk> tags.
