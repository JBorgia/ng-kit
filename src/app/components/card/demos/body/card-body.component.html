The card body has just 3 styles on the wrapper for untagged content that is transcluded into it:
{
 border-radius: 0 0 2px 2px;
 color: var(--text-color-primary);
 overflow: hidden;
}

The card body can be replaced by adding ngk-card-body or xmCardBody to any div.

In the example, as the xmCardBody directive is present the root level <article> tag the <div> tag is overridden and will not display. Checkout the code to see the missing <div> tag.
