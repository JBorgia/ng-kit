If you want to style the entire header area, content can be transcluded using the ng-kit-card-title, xmCardTitle, ng-kit-card-title-area, or xmCardTitleArea directive.

Notice how the blue area in the example takes up the whole height and extends 300px wide. The transcluded content is still verically aligned center and horizontally aligns left.

*NOTE: If an xmCardTitleArea tag is present the root level <header> tag is overridden and will not display. Also, some things like background and text color do still apply to the header area but can be overridden.
